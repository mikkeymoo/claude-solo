# /ship — Ship It

1. **Final test run** — run all tests; if anything fails, stop
2. **PR** — create with: what changed, why, test plan, breaking changes
3. **Merge** — squash WIP commits if needed
4. **CI** — confirm it passes (or confirm local tests if no CI)
5. **Post-ship** — note monitoring, feature flags, or follow-up tasks

End with: "Shipped."
